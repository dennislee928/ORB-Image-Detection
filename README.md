# ORB 圖片對比檢測工具

基於 ORB 與 AKAZE 雙引擎特徵比對的圖片相似度檢測系統，適用於從大量圖片中快速找出相似圖片。

## 功能特色

- **雙引擎比對**：結合 ORB（速度快）與 AKAZE（精確度高）兩種特徵偵測演算法
- **翻轉偵測**：自動偵測原圖、水平翻轉、垂直翻轉、180 度旋轉四種方向
- **多重 RANSAC**：支援多重幾何變換模型偵測，可辨識局部裁切或拼貼
- **智慧快取**：首次分析後將特徵快取至磁碟（`.orb_cache.dat`），後續檢索大幅加速
- **特徵熱點圖**：視覺化呈現兩張圖片的特徵對應關係與匹配熱區
- **中文完整支援**：檔名、路徑皆支援中文
- **高 DPI 支援**：針對 Windows 高解析度螢幕最佳化

## 系統需求

- Windows 10 以上
- 記憶體 4 GB 以上（建議 8 GB）
- 不需要網路連線，所有運算皆在本機執行

## 支援格式

`.jpg`、`.jpeg`、`.png`、`.webp`

## 快速開始

### 使用 EXE 執行檔（免安裝）

直接執行 `ORB系統EXE檔/ORB.exe` 即可啟動。

### 使用 Python 原始碼

```bash
pip install opencv-python numpy
python ORB.py
```

## 使用方式

1. **選擇基準原圖** — 選取要比對的基準圖片
2. **選擇比對資料夾** — 選取包含待比對圖片的資料夾
3. **執行檢索** — 系統自動比對並依相似度排序
4. **查看熱點圖**（選用）— 選取表格中的圖片，點擊紅色按鈕查看特徵對應視覺化

## 相似等級

| 等級 | 信心度 | 說明 |
|------|--------|------|
| 極高相似 | > 85% | 幾乎為同一張圖片 |
| 高度相似 | 60% ~ 85% | 大量共同特徵 |
| 中度相似 | 30% ~ 60% | 部分相似 |
| 無關 | < 30% | 無明顯關聯 |

信心度 ≥ 30% 的圖片會自動提取至結果資料夾 `V83_Result_[基準圖名稱]`。
